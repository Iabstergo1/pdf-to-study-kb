- [ ] **Step 2: 使用 pymupdf 提取粗页范围并按标题裁剪**

运行 Python 脚本：

```python
import fitz
import re

doc = fitz.open("books/博弈论白皮书/input/博弈论研究完全自学入门-自救白皮书.pdf")

# Step 2a: 提取粗页范围（第 21-22 页，0-indexed 20-21）
raw_text = ""
for page_num in range(20, 22):
    page = doc[page_num]
    raw_text += page.get_text() + "\n"
doc.close()

# Step 2b: 按标题边界裁剪
# 起点：2.1 到底什么是博弈
# 终点：2.2 参与者(Players)
start_pattern = r'.*2\.1\s+到底什么是博弈'
end_pattern = r'.*2\.2\s+参与者\s*\(Players\)'

start_match = re.search(start_pattern, raw_text)
end_match = re.search(end_pattern, raw_text)

if not start_match:
    raise ValueError("未找到起点标题 '2.1 到底什么是博弈'")

start_pos = start_match.start()
end_pos = end_match.start() if end_match else len(raw_text)

sliced_text = raw_text[start_pos:end_pos].strip()

# Step 2c: 保存 source slice
with open("books/博弈论白皮书/pipeline-workspace/staging/GTW-002-01/source-slice.md", "w", encoding="utf-8") as f:
    f.write("# GTW-002-01 原文片段\n\n")
    f.write("- 来源：博弈论研究完全自学入门-自救白皮书.pdf\n")
    f.write("- 粗页码范围：21-22\n")
    f.write("- 裁剪起点：2.1 到底什么是博弈\n")
    f.write("- 裁剪终点：2.2 参与者(Players) 之前\n\n")
    f.write("## 原文内容\n\n")
    f.write(sliced_text)

print(f"Source slice 已保存，长度：{len(sliced_text)} 字符")
```


- [ ] **Step 2: 按 review skill 进行审校**

遵循 `.claude/skills/section-lesson-review/SKILL.md` 的规则：

**审校维度：**

1. **忠实性**（PASS/WARN/FAIL）
   - 核心概念是否准确反映原文
   - 关键结论是否与原文一致
   - 是否存在凭空扩展

2. **可学习性**（PASS/WARN/FAIL）
   - "学习定位"是否清晰
   - "先记住的结论"是否有锚点作用
   - "直觉解释"是否易懂

3. **轻重分级**（PASS/WARN/FAIL）
   - importance A 是否合理
   - "必须掌握"与 importance 是否匹配

4. **回原文条件**（PASS/WARN/FAIL）
   - "何时回原文"是否明确
   - 公式风险是否正确标记（本节 low，应无特殊标记）

5. **结构完整性**（PASS/FAIL）
   - 12 个必备章节是否全部存在
   - frontmatter 必填字段是否完整

**决策规则（按优先级）：**

1. **reject**：忠实性 FAIL 或结构完整性 FAIL
2. **revise**：任意维度 FAIL 或 WARN > 3
3. **accept**：无 reject/revise 条件
